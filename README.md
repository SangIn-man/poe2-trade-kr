# PoE2 Trade Quick Search 🔍

패스 오브 엑자일 2 **한국 거래소** (poe.kakaogames.com)에서  
자주 검색하는 아이템 조건을 저장해두고 한 번에 재검색할 수 있는 Chrome 확장 프로그램입니다.

> ⚠️ 이 프로그램은 Grinding Gear Games(GGG) 및 poe.ninja와 무관한 **비공식 커뮤니티 도구**입니다.

---

## 📦 설치 방법

1. 이 페이지 우측 상단 **Releases** → 최신 버전의 `poe2-trade-vX.X.zip` 다운로드
2. 다운로드한 ZIP 파일 압축 해제
3. Chrome 주소창에 `chrome://extensions` 입력 후 이동
4. 우측 상단 **개발자 모드** 토글 켜기
5. **압축해제된 확장 프로그램을 로드합니다** 클릭
6. 압축 해제된 폴더 선택

> 💡 설치 후 Chrome 우측 퍼즐 아이콘(확장 프로그램)에서 고정해두면 편리합니다.

---

## 🚀 기본 사용법

### 1단계 - 거래소에서 아이템 검색
[한국 거래소](https://poe.kakaogames.com/trade2/search/poe2/Runes%20of%20Aldur)에 접속해서 원하는 옵션으로 아이템을 검색합니다.

### 2단계 - ⭐ 즐겨찾기 저장
검색 결과의 각 아이템 우측에 **⭐ 버튼**이 생깁니다.  
저장하고 싶은 아이템의 ⭐을 클릭하면 **속성 선택 창**이 열립니다.

### 3단계 - 저장할 속성 선택
아이템의 모든 속성이 목록으로 표시됩니다.  
**필요한 속성만 체크**하고 나머지는 해제한 뒤 저장하면, 선택한 속성 조건만으로 필터가 만들어집니다.

> 💡 거래소에서 2개 속성으로 검색했더라도 아이템 자체는 6개 속성을 가질 수 있습니다.  
> 저장 시 필요한 속성만 선택하면 불필요한 조건이 포함되지 않습니다.

### 4단계 - 사이드 패널에서 재검색
Chrome 우측 **사이드 패널**에 저장된 필터가 나타납니다.  
원하는 필터의 **🔍 검색 버튼**을 클릭하면 거래소에서 바로 재검색됩니다.

## ✨ 주요 기능 상세

### 🎯 검색 수치 실시간 조절

저장된 필터를 열면 각 속성의 **최솟값(min)**을 직접 수정할 수 있습니다.

- **마우스 휠** — 수치 위에 마우스를 올리고 휠을 굴리면 값이 1씩 올라가고 내려갑니다
- **직접 입력** — 숫자 칸을 클릭해서 원하는 값 입력
- **체크박스** — 속성 옆 체크를 해제하면 해당 조건이 검색에서 제외됩니다 (삭제 없이 임시 비활성화)

---

### 📁 빌드별 탭 관리

여러 캐릭터나 빌드를 **탭으로 분리**해서 필터를 관리할 수 있습니다.

- **+ 빌드 추가** — 새 빌드(캐릭터) 생성
- **장비 탭** — 방어구, 무기 등 장비 필터 모음
- **서판 탭** — 서판 필터 모음 (서판 종류별 서브필터 지원)
- **커스텀 탭** — 자유롭게 이름 지정해서 필터 분류 가능
- 빌드 이름 클릭 시 접기/펼치기 가능

---

### 💎 서판 서브필터

서판 탭 안에서 **서판 종류별로 필터링**할 수 있습니다.

| 탭 | 설명 |
|----|------|
| 전체 | 저장된 서판 필터 모두 표시 |
| 의식 | 의식 서판만 표시 |
| 심연 | 심연 보석만 표시 |
| 방사능 | 방사능 보석만 표시 |
| 감독관 | 감독관 보석만 표시 |
| 탐험 | 탐험 서판만 표시 |
| 사원 | 사원 보석만 표시 |
| 균열 | 균열 서판만 표시 |
| 환영 | 환영 서판만 표시 |

서판이 많아졌을 때 원하는 종류만 빠르게 찾을 수 있습니다.

### 💰 경제 탭 (poe.ninja 시세)

> 📊 시세 데이터는 [poe.ninja](https://poe.ninja) API를 통해 실시간으로 가져옵니다.

거래소를 나가지 않고 사이드 패널에서 바로 시세를 확인할 수 있습니다.

| 카테고리 | 내용 |
|----------|------|
| 커런시 | 디바인, 카오스 등 화폐 환율 |
| 에센스 | 에센스 종류별 시세 |
| 징조 | 징조 시세 |
| 환영 액체 | 환영 관련 아이템 시세 |
| 심연 | 심연 관련 시세 |
| 혈통 | 혈통 지원 보석 시세 |
| **탐험** | 탐험 관련 아이템 시세 (Gwennen 등) |
| 균열 | 균열 관련 시세 |
| 베리시움 | 베리시움 시세 |
| 룬 | 룬 시세 |

> 💡 **탐험 탭**은 탐험 콘텐츠에서 나오는 아이템들의 현재 시세를 보여줍니다.  
> 탐험 중 나온 아이템을 팔지, 사용할지 빠르게 판단하는 데 유용합니다.

### 📤 필터 내보내기 / 가져오기

저장한 필터와 빌드 구조를 **JSON 파일로 백업**하거나 다른 PC로 이전할 수 있습니다.

**내보내기**: 설정 탭 → 📤 필터 내보내기 → JSON 파일 저장  
**가져오기**: 설정 탭 → 📥 필터 가져오기 → JSON 파일 선택 후 방식 선택

가져오기 방식:
- **기존에 추가** — 기존 필터는 유지하고 새 필터만 추가 (중복 자동 제외)
- **완전 교체** — 기존 필터 모두 삭제하고 파일 내용으로 교체

---

### ⚙️ 설정

**설정 탭**에서 다음을 변경할 수 있습니다:

- **리그** — 현재 플레이 중인 리그 이름 설정 (예: Runes of Aldur, Standard)
- **결과 표시 수** — 검색 시 가져올 아이템 수 (5 / 10 / 20개)

## 🔄 업데이트 방법

1. [Releases](../../releases) 페이지에서 최신 버전 ZIP 다운로드
2. 기존 폴더의 파일들을 새 파일로 교체
3. `chrome://extensions` 에서 이 확장 프로그램의 **새로고침 버튼(🔄)** 클릭

> 저장된 필터 데이터는 Chrome 내부에 보관되므로 업데이트해도 사라지지 않습니다.

## ❓ 자주 묻는 질문

**Q. ⭐ 버튼이 안 보여요**  
A. 확장 프로그램 설치 후 거래소 페이지를 새로고침(F5) 해주세요.

**Q. 검색해도 결과가 안 나와요**  
A. 설정 탭에서 리그 이름이 현재 시즌과 일치하는지 확인해주세요.

**Q. Chrome 재시작 후 "개발자 모드 확장 프로그램" 경고가 떠요**  
A. 개발자 모드로 설치된 확장 프로그램의 정상적인 Chrome 경고입니다. **취소** 또는 **계속 사용**을 누르면 됩니다.

# PoE2 Trade Quick Search 🔍

Path of Exile 2 **한국 거래소**에서 아이템을 ⭐ 즐겨찾기로 저장하고, 페이지 사이드바에서 **원클릭으로 재검색**하는 Chrome 확장 프로그램입니다.

> ⚠️ **비공식 도구 안내**
> 이 확장은 **개인이 제작한 비공식 도구**이며, **Grinding Gear Games(GGG)** 및 **poe.ninja** 와 아무런 제휴·후원·연관 관계가 없습니다. Path of Exile 2 및 관련 자산의 모든 권리는 GGG에 있습니다. 본 도구 사용으로 발생하는 결과에 대한 책임은 사용자 본인에게 있습니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| ⭐ 즐겨찾기 저장 | 거래소 아이템 각 행에 ⭐ 버튼을 주입, 클릭 한 번으로 필터 저장 |
| 🧬 속성 자동 파싱 | 아이템의 속성·모드를 자동 추출하여 검색 가능한 필터로 변환 |
| 📉 최소값 자동 설정 | 파싱된 수치의 **기본 80% floor** 값을 최소값으로 자동 지정 (바로 쓸 수 있는 검색 기준) |
| 🔁 원클릭 재검색 | 사이드바에서 저장한 필터를 클릭 한 번으로 다시 검색 |
| 🗂️ 빌드별 탭 관리 | 빌드/용도별 탭으로 필터를 분류·관리 |
| 🧩 PoB 연동 | Path of Building 데이터와 연계하여 필터 활용 |
| 🪧 서판(Tablet) 서브필터 | 태블릿 전용 서브필터 지원 |
| 💰 경제(Economy) 탭 | poe.ninja 시세를 불러와 가격 참고 |
| 📤 내보내기 / 가져오기 | 필터 묶음을 JSON으로 내보내고 다시 가져오기 |
| 🎛️ 사이드바 사용자 설정 | 좌/우 위치 전환, UI 줌, 패널 너비를 사용자가 직접 조절 |

## 📦 설치 방법

Chrome 웹 스토어 등록 전이므로 **압축 해제(unpacked) 방식**으로 설치합니다.

1. **Releases** 페이지에서 `poe2-trade-v1.2.1.zip` 을 다운로드합니다.
2. 다운로드한 ZIP 파일의 **압축을 해제**합니다.
3. 브라우저 주소창에 `chrome://extensions` 를 입력해 확장 프로그램 페이지를 엽니다.
4. 우측 상단 **개발자 모드(Developer mode)** 를 **ON** 으로 켭니다.
5. **"압축해제된 확장 프로그램을 로드"(Load unpacked)** 버튼을 클릭합니다.
6. 2번에서 압축을 푼 **폴더를 선택**하면 설치가 완료됩니다.

> 💡 **Chrome 계열 브라우저 모두 지원** — Microsoft Edge(`edge://extensions`), Naver Whale(`whale://extensions`) 등 Chromium 기반 브라우저에서도 동일한 방법으로 설치·사용할 수 있습니다.

## 🚀 사용법

1. **한국 PoE2 거래소에서 아이템 검색**
   한국 거래소(`poe.kakaogames.com`)에서 평소처럼 원하는 아이템을 검색합니다.
2. **⭐ 버튼으로 필터 저장**
   검색 결과의 각 아이템 행에 표시된 **⭐ 버튼**을 클릭하면, 해당 아이템의 속성이 자동 파싱되어 필터로 저장됩니다. (수치 최소값은 기본 80% floor 로 설정됩니다.)
3. **사이드바 열기**
   페이지 우측에 표시되는 **인-페이지 사이드바**를 열어 저장된 필터 목록을 확인합니다.
4. **저장된 필터로 재검색**
   원하는 필터를 클릭하면 해당 조건으로 거래소를 **원클릭 재검색**합니다.
5. **필터 편집**
   필터 편집 모달에서 각 속성을 **on/off** 토글하거나 **최소값을 직접 조정**해 검색 정밀도를 높일 수 있습니다.

## 🔄 업데이트 방법

1. **Releases** 에서 최신 `poe2-trade-v1.2.1.zip` 을 다운로드하고 압축을 해제합니다.
2. 기존 확장 폴더를 새 폴더로 **교체**하거나, 새 폴더를 별도로 로드합니다.
3. `chrome://extensions` 에서 확장 프로그램의 **새로고침(↻)** 버튼을 클릭합니다.
4. 열려 있던 **거래 페이지를 새로고침**하면 변경 사항이 적용됩니다.

## ❓ 자주 묻는 질문

**Q. 글로벌(영문) 거래소에서도 되나요?**
A. 이 확장은 **한국 거래소 전용**으로 설계되었습니다. 핵심 기능(⭐ 저장 · 원클릭 재검색)은 한국 거래소(`poe.kakaogames.com`)에 맞춰 동작합니다.

**Q. 저장한 데이터는 어디에 보관되나요?**
A. 모든 필터·설정은 브라우저 내부의 `chrome.storage.local` 에 **로컬 저장**됩니다. 외부 서버로 전송되지 않습니다.

**Q. 사이드바가 보이지 않아요.**
A. 사이드바는 **거래소 페이지에서만** 표시됩니다. 거래소 페이지가 열려 있는지 확인하고, 그래도 보이지 않으면 **페이지를 새로고침**하세요. 확장을 새로 설치/업데이트한 직후라면 `chrome://extensions` 에서 새로고침 후 거래 페이지도 다시 불러오세요.

**Q. 어떤 브라우저에서 쓸 수 있나요?**
A. Chrome, Microsoft Edge, Naver Whale 등 **Chromium 기반 브라우저**에서 동일하게 사용할 수 있습니다.

## 📝 기타

- **한국 거래소 전용** 도구입니다.
- **Chrome 계열(Chromium) 브라우저** 를 지원합니다.
- 모든 데이터는 `chrome.storage.local` 에 **로컬 저장**되며 **외부로 전송되지 않습니다.**
- 시세 정보는 **poe.ninja** API를 사용합니다. 데이터 제공에 감사드립니다. (poe.ninja 와 본 도구는 무관합니다.)
- 본 도구는 **개인 제작 비공식 도구**이며 **GGG · poe.ninja 와 제휴 관계가 없습니다.** 모든 권리는 각 권리자에게 있습니다.

# PoE2 Trade Quick Search

패스 오브 엑자일 2 한국 거래소(poe.kakaogames.com)에서 즐겨찾기 필터를 저장하고 빠르게 재검색하는 Chrome 확장 프로그램입니다.

## 설치 방법

1. 이 폴더를 PC에 다운로드 (ZIP 압축 해제)
2. Chrome 브라우저에서 `chrome://extensions` 접속
3. 우측 상단 **개발자 모드** 토글 활성화
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 이 폴더 선택

## 사용 방법

1. [한국 거래소](https://poe.kakaogames.com/trade2/search/poe2/Runes%20of%20Aldur) 접속
2. 원하는 아이템 검색 후 결과에서 **⭐ 버튼** 클릭
3. 저장할 속성 선택 후 확인
4. Chrome 우측 사이드 패널에서 저장된 필터 관리 및 재검색

## 주요 기능

- ⭐ 거래소 검색 결과에서 즉시 필터 저장
- 📁 빌드별 탭 관리 (장비 / 서판 / 커스텀)
- 🔎 서판 종류별 서브필터 (의식 / 심연 / 방사능 / 감독관 / 탐험 / 사원 / 균열 / 환영)
- 💰 poe.ninja 환율 / 시세 조회
- 📤 필터 내보내기 / 가져오기 (빌드 구조 포함)

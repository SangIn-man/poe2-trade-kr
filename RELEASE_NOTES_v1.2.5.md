# v1.2.5

## 업데이트 내용

- 거래소 능력치 필터 검색을 거래소 기본 키워드 검색 방식으로 변경했습니다.
  - 기존 `키워드 ON/OFF` 토글과 전용 드롭다운을 제거했습니다.
  - 능력치 필터/능력치 그룹 검색 입력창이 열리면 검색어 앞에 `~`가 자동으로 붙습니다.
  - 사용자가 `~` 없이 입력하거나 붙여넣어도 자동으로 `~` 접두사가 유지됩니다.

- 경로석 정규식 생성기의 수치 조건을 개선했습니다.
  - 수치 조건은 OR 모드에서도 `|`로 묶지 않고 공백으로 연결해 AND 조건처럼 동작하게 했습니다.
  - `아이템 수량` 수치 조건을 제거했습니다.
  - `몬스터 효율`, `몬스터 희귀도`, `경로석 출현확률` 수치 조건을 추가했습니다.

## 참고

- 기존 저장 필터와 설정은 그대로 유지됩니다.
- 업데이트 후에는 `chrome://extensions`에서 확장 프로그램을 새로고침하고, 열려 있던 거래소 페이지도 새로고침해 주세요.

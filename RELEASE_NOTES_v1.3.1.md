# v1.3.1

## 업데이트 내용

- 정규식(미완성) 탭에 아래 **수치 조건 입력칸 5종**을 추가했습니다.
  - `아이템 희귀도`
  - `무리 규모`
  - `경로석 출현 확률`
  - `몬스터 효율`
  - `몬스터 희귀도`
- 경로석 정규식 수치 조건은 다음 **짧은 패턴** 기준으로 생성되도록 정리했습니다.
  - `아이템 희귀도: 귀도`
  - `무리 규모: ^무`
  - `경로석 출현 확률: 현.확`
  - `몬스터 효율: 몬.*효`
  - `몬스터 희귀도: 몬.*희`
- 거래소의 **현재 검색조건 저장** 기능에서 경로석/지도 전용 거래소 조건이 누락되던 문제를 수정했습니다.
- 능력치 검색창의 `~` **자동 접두사 방식**이 다시 기본 동작하도록 정리했습니다.

## 참고

- 업데이트 후에는 `chrome://extensions`에서 확장 프로그램을 새로고침하고, 열려 있던 거래소 페이지도 새로고침해 주세요.
- 기존 필터와 설정은 그대로 유지됩니다.

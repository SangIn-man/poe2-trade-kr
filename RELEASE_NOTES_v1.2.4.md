# v1.2.4

## 업데이트 내용

- 거래소 검색 UI에 **키워드 검색 토글** 버튼을 추가했습니다.
  - `History` 버튼 **왼쪽**에 `🔍 키워드 ON/OFF` 토글이 생깁니다.
  - **ON**: 스탯 검색창에서 우리만의 키워드 검색 드롭다운이 떠서 한글 키워드로 옵션을 빠르게 찾을 수 있습니다.
  - **OFF**: 거래소 **기본 검색 UI**를 그대로 사용합니다. (기본값: OFF)
  - 토글 상태는 저장되어 다음에 접속해도 유지됩니다.

## 참고

- 기존 v1.0 ~ v1.2.3에서 저장한 필터와 설정은 그대로 유지됩니다.
- 업데이트 후에는 `chrome://extensions`에서 확장 프로그램을 새로고침하고, 열려 있던 거래소 페이지도 새로고침해 주세요.

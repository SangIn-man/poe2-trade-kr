# v1.2.3

## 업데이트 내용

- 설정에 **다른 페이지 허용** 옵션을 추가했습니다.
  - 켜두면 거래소가 아닌 일반 웹페이지에서도 사이드바를 열 수 있습니다.
  - `file://` 페이지는 브라우저 확장 프로그램 설정에서 파일 URL 접근 허용이 필요할 수 있습니다.
- 사이드바 표시와 스크롤 동작을 개선했습니다.
  - 브라우저 줌, 해상도, 작업표시줄 높이에 따라 패널 하단이 잘리거나 여백이 과하게 생기던 문제를 줄였습니다.
  - TradeQuick, Economy, Utility 탭 내부가 각각 자연스럽게 스크롤되도록 조정했습니다.
- 에러 로그 확인 기능을 개선했습니다.
  - 설정 탭에서 최근 에러를 확인하고, 복사하거나 TXT로 저장할 수 있습니다.
- Utility 탭을 추가했습니다.
  - 기존 Expedition 기능은 Utility 안으로 이동했습니다.
  - `정규식(미완성)` 탭을 추가했습니다.
- `정규식(미완성)` 탭에 경로석 정규식 생성기를 추가했습니다.
  - 옵션 검색, 포함/제외 선택, 수량·희귀도·무리 규모 조건 입력, 프리셋 저장을 지원합니다.
  - 정규식 기능은 아직 실험 중이라 일부 조건은 인게임 검색 결과와 다를 수 있습니다.
- 저장된 필터 카드에서 사용성이 낮던 `KR거래소` 버튼을 제거했습니다.

## 참고

- 기존 v1.0 ~ v1.2.2에서 저장한 필터와 설정은 그대로 유지됩니다.
- 업데이트 후에는 `chrome://extensions`에서 확장 프로그램을 새로고침하고, 열려 있던 거래소 페이지도 새로고침해 주세요.
